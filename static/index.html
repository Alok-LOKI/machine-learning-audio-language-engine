[ SYSTEM ARCHITECTURE & METHODOLOGY ]

1. DUAL ENCODER PARADIGM

Our Audio-Language Engine utilizes a weakly supervised dual-encoder contrastive learning framework. By learning from noisy web captions (WavCaps), we encode raw waveforms into a shared multimodal acoustic-semantic latent space.

2. COMPONENT HIGHLIGHTS

  • Audio Encoder: Spectrogram-based CNN/Transformer (HTSAT) extracting frame-level acoustic features mapped to an aggregate embedding vector.
  • Text Encoder: Transformer-based model (e.g., BERT/RoBERTa) producing sequence-pooled text embeddings.
  • Objective Function: InfoNCE (Normalized Temperature-scaled Cross Entropy) loss to maximize cosine similarity of matched audio-text pairs while pushing negative pairs apart.
